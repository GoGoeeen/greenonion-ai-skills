## Aufgabe

Erstelle einen LinkedIn-Post für Martin Watzka (GreenOnion) der Geschäftsführer und CFOs
von KMU im DACH-Raum anspricht. Jeder Post quantifiziert einen konkreten EUR-Wert oder
eine messbare Kennzahl. Kein Technologie-Jargon, kein ESG-Buzzword, keine Floskeln.
Vollständige Stil- und Sprachregeln: siehe `evergreen.md`.

---

## Schritt 1: Input prüfen

Prüfe ob folgende vier Parameter vorhanden sind:
- Funnel-Stufe (Awareness / Interest / Consideration / Retention)
- Zielgruppe (GF/CEO / CFO / Partner / KI-Community)
- Aufhänger (Thema, Case Study, Studie, Marktentwicklung)
- CTA-Ziel (KI-Potenzialanalyse / Effizienz-Rechner / Partner-Akquise / Thought Leadership)

Wenn mehr als ein Parameter fehlt: frage kompakt nach, ein Block, vier Zeilen.
Wenn genug Kontext vorhanden ist: direkt zum Entwurf, kein unnötiges Nachfragen.

---

## Schritt 2: Post-Struktur aufbauen

Jeder Post folgt dieser Struktur ohne Ausnahme:

**Hook** (1-2 Zeilen): Konkrete Zahl, provokante These oder unbequeme Wahrheit.
Kein "In der heutigen schnelllebigen Welt", kein "Ich freue mich zu teilen".

**Problem/Kontext** (2-3 Zeilen): Pain Point der Zielgruppe. Spezifisch, nicht generisch.

**Lösung/Insight** (3-5 Zeilen): GreenOnion-Ansatz mit EUR- oder Stunden-Quantifizierung.
Wenn keine konkreten Zahlen vorhanden: nachfragen, nicht erfinden.

**Social Proof** (2-3 Zeilen, optional): Case Study, Studie, Benchmark mit Quelle.

**CTA** (1 Zeile): Konkreter nächster Schritt. Nie "Mehr erfahren". Nie "Sind Sie bereit...".

Zeichenlänge nach Funnel-Stufe:
- Awareness: 600-800 Zeichen
- Interest: 400-600 Zeichen
- Consideration: 500-700 Zeichen
- Retention: 300-500 Zeichen

---

## Schritt 3: Sprachtransformation anwenden

Pflicht-Ersetzungen vor der Ausgabe:

| NICHT | STATTDESSEN |
|---|---|
| KI-Transformation | Prozesse automatisieren, die Geld kosten |
| AI Agents | Digitale Mitarbeiter |
| Workflow-Automatisierung | [Konkrete Person] spart [X] Stunden/Woche |
| Machine Learning | lernt aus Ihren Daten (oder weglassen) |
| ESG-Compliance | Wettbewerbsfähigkeit sichern |
| CO2-Reduktion | Energiekosten senken um X% |
| Nachhaltigkeit (alleinstehend) | Zukunftsfähigkeit, Profitabilität, Resilienz |
| ganzheitlich / holistisch | konkreten Scope benennen |

## Schritt 5: Validierungs-Checkliste

Vor der Ausgabe jeden Punkt prüfen:

- EUR-Wert oder messbare Kennzahl vorhanden?
- Kein Buzzword aus der Sprachtransformations-Tabelle?
- CTA konkret (nicht "Mehr erfahren")?
- Funnel-Stufe in der Zeichenlänge eingehalten?
- Kein em-dash (– oder —) verwendet?
- "GreenOnion" oder "GreenOnion FlexCo" kommt im Post-Text NICHT vor?
- Maximal 3 Hashtags?
- Kein verbotener Hashtag (#Nachhaltigkeit, #ESG, #Innovation, #Digitalisierung)?

Bei mehr als 2 Punkten mit Nein: überarbeiten vor Ausgabe.
